Google Scholar LinkedIn Github Stackoverflow ResearchGate Orcid

Je suis docteur en informatique, diplômé de l’Université de Lille (Cristal / Inria).

Je suis aujourd’hui Data Scientist dans la startup Diag n’ Grow. Mes travaux portent sur les données et l’intelligence artificielle dans un contexte d’actif immatériel avec des sujets variés comme la classification de brevets par exemple.

J’ai été le porteur du projet de startup Optimo à l’Inria Startup Studio de septembre 2021 à janvier 2023.

Intérêts et projets actuels:

  • Optimisation Multi-objectif
  • Algorithme MOEA/D
  • Data Science
